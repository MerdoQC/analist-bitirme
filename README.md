📌 Project Overview

This project analyzes the factors that affect student academic performance using a complete data analytics workflow.

The objective of the project is to identify which variables have the strongest impact on student success and present these insights through data analysis and interactive visualizations.

The project demonstrates an end-to-end data pipeline, including:

Dataset acquisition

Data storage and cleaning

Exploratory data analysis (EDA)

Insight generation

Dashboard visualization

📊 Dataset

The dataset used in this project contains information about student demographics, study habits, and academic performance indicators.

Dataset includes variables such as:

Study time

Parental education

Internet access

Absences

Educational support

Final grades
